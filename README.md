# def_ref_mch_manager
Blender 5.1 addon that helps organize major bone types.

<img width="270" height="265" alt="image" src="https://github.com/user-attachments/assets/8071629c-ed05-4fca-9d6a-d7e01d7bdda9" />

Ever wanted to skip making th ereference layer? Ever got anoyed at selected layered mechanical bones? Here my solution!
All you have to do is set the prefixes you use for each bone type and hit organize and this addon will not only add all the bones to a corisponding bone layer, but also set the colour and shape for you.
You can even press def to ref to convert you defformation skeleton to a reffence rig in seconds!
